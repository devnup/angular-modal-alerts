Tutorial: Elasticsearch Util - Getting Started

Elasticsearch Util - Getting Started

The Elasticsearch Util is a wrapper for the official Elasticsearch Nodejs Module. It handles the module startup and configuration definitions, getting the Bonsai Host URL from the environment variables or from default.json configuration file.


Useful links

  • Official documentation
  • Github repository
  • NPM module page


Getting started

To use the module simply import the file using require.

var els = require('./path/to/els');

// Send ping request to server
els.ping({
  requestTimeout: 1000,
  hello: "elasticsearch!"
}, function (error) {
  console.log(error || "Ping successfully sent!");
});


Creating an index

To create an index, use the method create

els.indices.create({
  index: 'test'
}, function (err, result) {
  console.log(err || result);
});
